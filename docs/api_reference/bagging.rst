Bagging Module
==============

Time-series-aware bagging with bootstrap strategies.

Provides moving block bootstrap, stationary bootstrap, and feature bagging
strategies that preserve temporal dependence structure.

.. seealso::

   :doc:`/api/bagging`
      User guide with detailed usage examples.

   `Politis & Romano (1994) <https://www.jstor.org/stable/2291225>`_
      Original stationary bootstrap paper.
