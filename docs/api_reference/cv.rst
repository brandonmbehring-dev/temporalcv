Cross-Validation Module
=======================

Walk-forward and cross-fitting cross-validation for time series.

Provides sklearn-compatible cross-validators that respect temporal ordering
and enforce gap between training and test periods.

.. seealso::

   :doc:`/api/cv`
      User guide with detailed usage examples.

   :class:`sklearn.model_selection.TimeSeriesSplit`
      sklearn's basic time series splitter (less feature-rich).
