Inference Module
================

Wild cluster bootstrap for inference on cross-validation fold statistics.

Standard clustered standard errors are unreliable with few clusters
(typical CV has 5-10 folds). The wild bootstrap provides valid inference
in this setting.

.. seealso::

   :term:`wild bootstrap`
      Glossary entry for wild cluster bootstrap.

   `Cameron et al. (2008) <https://doi.org/10.1162/rest.90.3.414>`_
      Theoretical foundations for wild cluster bootstrap.
