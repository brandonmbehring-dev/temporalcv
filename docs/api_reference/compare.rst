Compare Module
==============

Model comparison adapters for external forecasting libraries.

Provides adapters for comparing sklearn models against statistical
forecasting baselines from statsforecast and other libraries.

.. note::

   Requires optional dependencies:

   .. code-block:: bash

      pip install temporalcv[compare]
